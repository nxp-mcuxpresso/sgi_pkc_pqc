Parameters
[out]pSignatureModePointer to a mode descriptor that will be updated by this function.
[in,out]pProtocolDescriptorPointer to an ML-DSA protocol descriptor that will be updated by this function.
[in]optionsStruct type used to indicate the ML-DSA mode (44, 65, 87), the API mode (PURE and PRE-HASH) and whether internal or external API is expected.
[in]pPreHashAlgoIf PRE-HASH is used, the algorithm that is used to pre-hash the message; otherwise NULL. Note that this is a void pointer: it will either accept mcuxClHash_Algo_t or mcuxClXof_Algo_t.
[in]ctxUser provided context string; otherwise NULL.
[in]ctxLengthLength of the user provided context string; otherwise 0U.
