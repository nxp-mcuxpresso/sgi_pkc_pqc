const mcuxClSignature_ModeDescriptor_t mcuxClSignature_ModeDescriptor_ECDSA
extern

ECDSA Signature mode descriptor.

This mode requires that mcuxClSignature_verify is called with additional protection on pIn and inSize parameters. mode, pIn and inSize parameters must be protected using mcuxClSignature_verify_recordParam function before executing mcuxClSignature_verify.
