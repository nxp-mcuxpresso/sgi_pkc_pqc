◆ MCUXCLBUFFER_UPDATE

#define MCUXCLBUFFER_UPDATE ( name,
offset )

Update the buffer pointer with the given offset.

Warning
When used inside of a function, MCUXCLBUFFER_UPDATE may have side effects outside of the function call. Therefore, updating a buffer that is passed as function parameter should be avoided. Instead, MCUXCLBUFFER_UPDATE should be used on temporary buffers that have been initialized within a function, or derived from another buffer.
Parameters
namethe name of the buffer variable to update.
offsetthe amount of bytes that the buffer pointer should be moved.
